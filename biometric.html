Biometric Iris Identification

Developing high quality biometric authentication through an Android application for iris capture and recognition.

Project Description:

Duration: May 2023 - Jul 2023

Team: Natalie Zhu, Naveenkumar Venkataswamy, Dr. Masudul Haider Imtiaz

I interned at Clarkson Universityâ€™s AI Vision Lab (AVHBAC) to further their work on developing biometric sensors, biometric data acquisition systems, and biometric testing methodologies. One particular goal in this area is to achieve high quality image capture of iris data, which would result in achieving an optimal biometric authentication system.

This project aimed to develop a mobile camera-based Android application to perform accurate image capture and storage of irises for users to utilize biometric identification. I decided to improve the performance and usability of this Android application and help develop a new quality assessment feature.

Project Achievements:

> Developed new image quality assessment feature through utilizing OpenCV to evaluate the sharpness of raw image bitmaps.

> Restructured and optimized Java codebase, significantly reducing code complexity and enhancing maintainability.

> Enhanced user interface and experience by redesigning application front-end.

Technologies Used:

> Android App Development Framework: Android Studio & Android SDK

> OpenCV library

> Java, HTML, CSS

Further Project Details

Image quality assessment feature:

> Implemented image blurriness check with the OpenCV library, applying the Laplacian method in Java.

> After assessment of bluriness threshold, provided user feedback on image quality.

> Optimized this process of image capture, rotation, analysis, and storage.

App Optimization:

> Moved time-consuming operations to background threads.

> Restructured code to follow object-oriented principles and reduce resource consumption
